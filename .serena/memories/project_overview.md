## Purpose
ExoChat is an **open-source, self-hosted, end-to-end encrypted chat platform**. 
Think WhatsApp but fully self-hosted, GDPR/NIS2 compliant, and open-source.

## Key Features
- 1-on-1 and group chats
- End-to-end encryption (Signal Protocol, configurable per conversation)
- Voice & video calls (WebRTC via LiveKit)
- File & image sharing (stored in MinIO)
- Typing indicators, read receipts, online status
- Message search with configurable retention
- GDPR compliance (data export, account deletion, consent management, audit logging)
- NIS2 compliance (security event logging, breach notification tooling)
- Self-hosting via single `docker-compose up`
- Setup wizard on first run

## Target Audience
- Privacy-conscious users and organizations
- Companies needing GDPR-compliant chat (especially in Germany/EU)
- Self-hosting enthusiasts
